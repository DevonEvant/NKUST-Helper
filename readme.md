## account
學生帳號
- 管理登入(✓)
- 登出(✓)
- 儲存帳號密碼及與帳號有關的cookies(✗手機端完成)

## bus
校內公車 - _**ignore**_
- 查詢會員狀態(✓) - queryMemberState()
- 查詢可預約車輛(✓) - queryWithDate()
- 預約搭車(✓) - reserve()
- 取消預約搭車(✓) - cancelReserve()
- 查詢預約紀錄(✓) - queryReserved()
- 查詢搭車記錄(✗)
- 查詢罰鍰(✗)

## classInfo

- 成績(✓) - getGrade()
- 成績預警(✓) - earlyGradeWarning()
